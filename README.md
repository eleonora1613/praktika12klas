# FDMC & Smart Farming System Практически Задачи

Това репозитори съдържа два учебни проекта, разработени като част от практическото обучение в 12. клас:

- 🐱 **FDMC** – уеб приложение за управление на котки
- 🌱 **Smart Farming System** – симулационен хардуерен проект, разработен в Tinkercad

---

## 🐱 FDMC – Web Application

## Стартиране на проекта

FDMC представлява ASP.NET Core MVC уеб приложение, което използва SQL Server база данни (LocalDB).

За да бъде стартирано успешно, са необходими следните стъпки:

### Необходими инструменти

- Visual Studio (с ASP.NET и Web Development workload)
- .NET SDK (съвместима версия с проекта)
- SQL Server LocalDB (вграден в Visual Studio)

---

## Отваряне на проекта

1. Клонирай репозитори:

git clone <линк към репозитори>

или го изтегли като ZIP файл и го разархивирай.

2. Отвори файла:

FDMC.sln

в Visual Studio.

---

## Настройване на базата данни

Проектът използва SQL Server LocalDB.

Connection string се намира в:

appsettings.json

Примерна конфигурация:

"ConnectionStrings": {
  "DefaultConnection": "Server=(localdb)\\MSSQLLocalDB;Database=FDMC;Trusted_Connection=True;"
}

---

## База данни

В архива на проекта е включен SQL файл:

FDMC.sql

Този файл съдържа цялата структура на базата данни, включително:

- създаване на таблица Cats
- всички необходими полета и ограничения
- примерни записи за тестване
- конфигурация за първоначално зареждане на данните

---

## Възстановяване на базата данни

За да се зареди базата:

1. Създай нова база данни в SQL Server LocalDB с име:
   FDMC

2. Отвори файла:
   FDMC.sql

3. Изпълни скрипта (Run / Execute)

След това базата ще бъде напълно готова за използване с приложението.

---

## Стартиране на приложението

След конфигурация:

1. Стартирай проекта с Ctrl + F5  
2. Приложението ще се отвори в браузър  
3. Достъп до функционалности:
   - Добавяне на котка
   - Редактиране
   - Изтриване
   - Преглед на детайли

---

## ⚠️ Важно

- Базата данни трябва да бъде налична преди стартиране
- Приложението работи с локална база (LocalDB)
